## ⚛️ React {#react}

[![Astro 3.0](/emails/issues/158/astro3.jpg)](https://astro.build/blog/astro-3/)

[**Astro 3.0**](https://astro.build/blog/astro-3/)

Cette nouvelle version d'Astro innove notamment par l'utilisation de la nouvelle API View Transition, et la capacité à naviguer de manière fluide et animée d'une page à l'autre, tout en préservant l'état des composants affichés. Cela réduit le besoin d'une SPA dans de nombreux cas d'usage.

Autres améliorations notables :

- Optimisation d'image stable
- Performances de rendu plus rapides : les composants Astro rendent 30 à 75 % plus vite.
- Améliorations SSR pour Serverless : De nouvelles façons de se connecter à votre plateforme d'hébergement.
- Améliorations HMR pour JSX : Prise en charge de Fast Refresh pour React et Preact.
- Sortie de build optimisée : HTML plus propre et plus performant.

## 📱 React-Native {#react-native}

[![VisionCamera 3.0](/emails/issues/158/visioncamera3.jpg)](https://github.com/mrousavy/react-native-vision-camera/releases/tag/v3.0.0)

[**VisionCamera 3.0**](https://github.com/mrousavy/react-native-vision-camera/releases/tag/v3.0.0)

La nouvelle version de VisionCamera vient de sortir. Points forts :

- Réécriture complète d'Android Camera2
- Pipeline vidéo OpenGL GPU personnalisé
- Nouvelles fonctionnalités de caméra
- Nouveau moteur de traitement d'images (react-native-worklets-core)
- Nouvelle API de plugin de traitement d'images

Marc a pris la décision de finalement [supprimer l'intégration avec Skia](https://github.com/mrousavy/react-native-vision-camera/pull/1740) pour maintenir une taille de code et une complexité raisonnables sur VisionCamera.

Dans [**Détection de pose avec VisionCamera V3, TFLite et Skia**](https://mrousavy.com/blog/VisionCamera-Pose-Detection-TFLite), il explique ce qui est possible avec une telle intégration. Les résultats sont très impressionnants.

[![Détection de pose avec VisionCamera V3, TFLite et Skia](/emails/issues/158/frame-processing-skia.jpg)](https://mrousavy.com/blog/VisionCamera-Pose-Detection-TFLite)
