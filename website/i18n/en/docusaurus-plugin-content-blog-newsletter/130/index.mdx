## 📱 React-Native {#react-native}

[![Tamagui 1.0](/emails/issues/130/tamagui.jpg)](https://tamagui.dev/blog/version-one)

**[Tamagui 1.0](https://tamagui.dev/blog/version-one)**

Tamagui est officiellement sorti de la version bêta et facilite la création de projets multiplateformes (web et mobile) basés sur la même base de code en React, sans compromettre la DX ou les performances. Le projet est divisé en 3 parties :

- Tamagui Core : un runtime CSS-in-JS basé sur React-Native et [React-Native-Web-Lite](https://www.npmjs.com/package/react-native-web-lite)
- Tamagui Static : un compilateur basé sur Babel et une pré-évaluation Node qui prend en charge l'aplatissement des vues et l'extraction de styles CSS atomiques
- Tamagui UI : une bibliothèque de composants React et de hooks pour faciliter l'utilisation

Tamagui est un projet innovant salué par les leaders d'opinion en matière de stylisation et de multiplateforme. Il peut s'avérer intéressant même si vous ne ciblez qu'une seule plateforme. La prise en charge des fonctionnalités de React 18 comme les Server Components et le Concurrent Mode est déjà intégrée. Cela vaut le coup d'essayer !

[![React Native Skia : Rétrospective d'une année et perspectives](/emails/issues/130/skia.jpg)](https://shopify.engineering/react-native-skia-2022)

**[React Native Skia : Rétrospective d'une année et perspectives](https://shopify.engineering/react-native-skia-2022)**

William Candillon propose une belle rétrospective du projet React-Native-Skia. Pour rappel, [Skia](https://skia.org/) est le moteur 2D de Google, utilisé par Chrome, Android, Flutter... Les performances ont été significativement améliorées avec le rendu sur le thread UI et une nouvelle API déclarative Skia DOM. La bibliothèque est flexible et extensible (Skottie, traitement d'images VisionCamera). En 2023, on peut s'attendre à des animations via les worklets, une intégration renforcée avec Reanimated ou la mise en page de texte.

- 📜 [First-class Support for TypeScript](https://reactnative.dev/blog/2023/01/03/typescript-first): React-Native 0.71 should be released very soon with full TypeScript support by default: type definitions, documentation and starter template.
- 📜 [Shopify - Migrating our Largest Mobile App to React Native](https://shopify.engineering/migrating-our-largest-mobile-app-to-react-native): Shopify is in the process of incrementally migrating their main app to React-Native. It's a 6 years old app with more than 300 screens, with no particular technical debt. A very interesting feedback on this ongoing migration (still in the beginning, around 10% done). The first approach (new features in React-Native) had not worked. They finally opted to convert the root of the app first.
- 📜 [Implementing CSS Style Inheritance in React Native](https://www.builder.io/blog/implementing-css-style-inheritance-in-react-native): explains how to reproduce the CSS cascade in React-Native, a CSS behavior not implemented for good reasons. This is especially useful for integration with a CMS, which often assumes that you can apply text styling to a parent View, like it's possible on the web.
