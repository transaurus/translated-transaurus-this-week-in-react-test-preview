## ⚛️ React {#react}

[![React Router 7.3 - Middleware de route](/emails/issues/225/rr.jpg)](https://github.com/remix-run/react-router/blob/main/CHANGELOG.md#v730)

**[React Router 7.3 \- Middleware de route](https://github.com/remix-run/react-router/blob/main/CHANGELOG.md#v730)**

Cette version de React Router introduit une nouvelle fonctionnalité majeure de middleware. Elle n'est pas encore prête pour la production, mais vous pouvez l'activer avec le drapeau expérimental `future.unstable_middleware`. Utilisable en mode framework ou bibliothèque, elle distingue middleware client et serveur. Idéal pour implémenter journalisation, authentification et autres préoccupations transversales.
