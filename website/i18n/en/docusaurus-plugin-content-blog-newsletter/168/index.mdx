## ⚛️ React {#react}

[![TanStack Router 1.0](/emails/issues/168/tanstack-router.jpg)](https://tanstack.com/router/v1)

**[TanStack Router 1.0](https://tanstack.com/router/v1)**

Le très attendu TanStack Router est sorti en version stable 1.0 avec une belle [🐦 bande-annonce](https://twitter.com/tannerlinsley/status/1738709130213560497), juste à temps pour Noël. Ce nouveau routeur repousse les limites du possible en matière de typage fort, de gestion d'état des URLs (paramètres de recherche) et de mise en cache, entre autres.

Le [guide SSR](https://tanstack.com/router/v1/docs/guide/ssr) mentionne un package `@tanstack/react-start`, probablement un teaser pour un futur [🐦 framework TanStack Start](https://twitter.com/sebastienlorber/status/1742589454563369123).

Autres packages TanStack récents que vous auriez pu manquer :

- 📦 [TanStack Config - Outil pour publier et maintenir des packages JavaScript de haute qualité](https://github.com/TanStack/config)
- 📦 [TanStack Store - Store agnostique aux frameworks, typé et avec des adaptateurs réactifs](https://tanstack.com/store/latest)
