## 💸 Commanditaire {#sponsor2}

[![Surveyjs - Bibliothèques JavaScript extensibles pour les enquêtes et formulaires](/emails/issues/136/surveyjs.jpg)](https://surveyjs.io/?utm_source=thisweekinreact&utm_medium=referral)

**[Bibliothèques JavaScript extensibles pour les enquêtes et formulaires](https://surveyjs.io/?utm_source=thisweekinreact&utm_medium=referral)**

[SurveyJS](https://surveyjs.io/form-library?utm_source=thisweekinreact&utm_medium=referral) est un ensemble de bibliothèques JavaScript open-source pour formulaires que vous pouvez facilement intégrer à votre application web. Il vous permet de **créer et exécuter plusieurs formulaires**, en conservant toutes les données sensibles sur vos serveurs, ou d'avoir **votre propre éditeur de formulaires WYSIWYG auto-hébergé** que les utilisateurs non techniques peuvent utiliser directement.

Vous pouvez également **analyser les résultats des enquêtes** dans des tableaux de bord personnalisés, sauvegarder vos formulaires au format PDF, ou simplement **les convertir en fichiers PDF modifiables**. Il offre une prise en charge native pour le **rendu React et Angular** et une liberté totale de choix pour le backend, car **toute combinaison serveur + base de données est entièrement compatible**.

## 🔀 Autre {#other}

[![Rspack](/emails/issues/138/rspack.jpg)](https://www.rspack.dev/)

**[Rspack](https://www.rspack.dev/)**

Annonce officielle du bundler Rust de ByteDance, compatible avec Webpack mais beaucoup plus rapide. Il sera difficile de choisir entre Rspack et Turbopack, qui a également annoncé [une prise en charge imminente des loaders Webpack](https://twitter.com/jaredpalmer/status/1619071988181651456). Notez que [Unplugin 1.3](https://twitter.com/antfu7/status/1634517321392726017) prend déjà en charge Rspack et permet de créer des plugins de bundler agnostiques. Il n'y a pas encore de support Unplugin pour Turbopack, mais j'espère que c'est prévu !

Lisez aussi [Rspack : Le bundler Web basé sur Rust qui allie haute performance et interopérabilité](https://valor-software.com/press-release/rs-pack-rust-based-web-bundler)
