## 📱 React-Native {#react-native}

[![État de React Native 2024 - Résultats de l'enquête](/emails/issues/223/state-of-rn.jpg)](https://results.stateofreactnative.com/)

**[État de React Native 2024 \- Résultats de l'enquête](https://results.stateofreactnative.com/)**

Les résultats de l'enquête sont disponibles et je suis extrêmement fier que parmi toutes les ressources React Native, cette newsletter ait obtenu le score de positivité le plus élevé. Merci ❤️ !

Je ne peux pas résumer toute l'enquête, mais voici quelques éléments que j'ai découverts :

- 61 % des développeurs RN viennent de React (web)
- iOS et Android restent les cibles RN les plus courantes, mais le Web représente également 22 % du marché
- Les bibliothèques avec le sentiment le plus positif sont : Reanimated, TanStack Query, Zustand, React Navigation, Expo Router, Skia.
- Pas de gagnant clair en matière de style. Les développeurs apprécient les API intégrées comme StyleSheet et les styles inline.
