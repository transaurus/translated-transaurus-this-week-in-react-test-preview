## 📱 React-Native {#react-native}

Les actualités React Native de cette semaine sont compilées par Benedikt. Commentaires, suggestions ou actualités à partager ? Contactez-moi sur [Twitter](https://twitter.com/bndkt).

[![Expo SDK 50](/emails/issues/171/expo50.jpg)](https://expo.dev/changelog/2024/01-18-sdk-50k)

**[Expo SDK 50](https://expo.dev/changelog/2024/01-18-sdk-50)**

Expo SDK 50 introduit les plugins Expo Dev Tools, propose de nouvelles API SQLite et Camera, et ajoute `@expo/fingerprint`, une nouvelle méthode pour déterminer si une mise à jour du bundle JS est compatible avec une build native donnée. Cette version marque également la [fin du support Webpack](https://blog.expo.dev/webpack-support-in-expo-cli-is-now-deprecated-e9831d7eb631) dans Expo, désormais déprécié au profit de Metro. Le moyen le plus pratique de se familiariser avec Expo 50 est probablement de regarder Simon le parcourir en 15 minutes sur [YouTube](https://www.youtube.com/watch?v=shaGaiwu4Fg).

**[Expo Router v3](https://expo.dev/changelog/2024/01-23-router-3)**

Parallèlement à Expo SDK 50, Expo Router v3 a été publié. Il apporte d'importantes améliorations de performance et stabilité ainsi qu'une meilleure documentation. On note notamment le support expérimental pour créer des endpoints serveur universels avec API Routes. Expo Router devient ainsi le tout premier framework React full-stack universel !
