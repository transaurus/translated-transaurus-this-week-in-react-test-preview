**[Vite 4](https://vitejs.dev/blog/announcing-vite4.html)**

Vite, outil frontend en pleine croissance, utilise désormais Rollup 3, simplifiant la gestion des ressources internes. Deux plugins React aux compromis distincts :

- [vite-plugin-react](https://github.com/vitejs/vite-plugin-react) : esbuild + Babel
- [vite-plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) : remplace Babel par SWC en développement, améliorant le démarrage à froid et le HMR

[![Codux - IDE visuel pour projets React](/emails/issues/128/codux.jpg)](https://dev.to/codux/introducing-codux-15j5)

**[Codux - IDE visuel pour projets React](https://dev.to/codux/introducing-codux-15j5)**

Nadav, co-fondateur de Wix, nous présente son nouveau projet : un IDE visuel pour React (et bientôt d'autres frameworks). Il ne s'agit pas de remplacer votre IDE actuel, mais plutôt de le compléter. Codux permet de travailler sur un composant de manière isolée, y compris la modification des props et styles via une interface visuelle. La vidéo [Meet Codux](https://www.youtube.com/watch?v=KG6i6cCz0P0) vous aidera à mieux comprendre. C'est un peu comme Storybook, mais avec la possibilité d'éditer et une synchronisation bidirectionnelle avec votre code.
