## ⚛️ React {#react}

[![TanStackDB](/emails/issues/234/tanstackdb.jpg)](https://github.com/TanStack/db)

**[TanStack DB](https://github.com/TanStack/db)**

TanStack 团队与 ElectricSQL 合作推出了全新库，带来诸多激动人心的功能：极快的查询引擎、细粒度响应式设计以最小化重渲染、健壮的事务原语以及默认的标准化数据！

该库构建于 TanStack Store 之上，并与 TanStack Query 协同工作，通过集合、实时查询和乐观更新扩展了功能。
