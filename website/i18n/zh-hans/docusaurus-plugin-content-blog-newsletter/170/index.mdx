## ⚛️ React {#react}

[![Waku 0.19](/emails/issues/170/waku.jpg)](https://waku.gg/blog/introducing-create-pages)

**[Waku 0.19](https://waku.gg/blog/introducing-create-pages)**

此版本引入了一个新的 `createPage` API，用于以编程方式创建布局和页面。这显著标志着Waku从作为React Server Components的学习体验转变为真正的生产就绪的React框架。预计在v0.20中会有一个轻量级的基于文件的路由。

**[美味的甜甜圈组件](https://frontendatscale.com/blog/donut-components/)**

这篇愉快而精彩的交互式文章使用甜甜圈类比（带有“children”洞的客户端组件）来解释客户端/服务器组件如何交错。它还涵盖了更高级的RSC组合主题。今天我学到了那个花哨的服务端动作渲染prop技巧！
