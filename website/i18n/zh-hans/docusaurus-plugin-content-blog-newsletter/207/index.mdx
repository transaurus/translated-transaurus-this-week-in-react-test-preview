## 📱 React-Native {#react-native}

[![默认启用新架构、开发工具等](/emails/issues/207/newarch.jpg)](https://reactnative.dev/blog/2024/10/23/release-0.76-new-architecture)

[**React Native 0.76 - 默认启用新架构、开发工具等**](https://reactnative.dev/blog/2024/10/23/release-0.76-new-architecture)

React Native 0.76 发布了重大更新！新架构现已默认启用，新增了 React Native DevTools 以改进调试，并支持 CSS 的 `boxShadow` 和 `filter` 属性以增强 UI 效果（仅在新架构中可用）。请注意一些破坏性变更：最低 SDK 要求已提升至 iOS 15.1 和 Android SDK 24（Android 7）。是时候升级并体验了！

v0.76 很酷，但更棒的是[新架构已到来](https://reactnative.dev/blog/2024/10/23/the-new-architecture-is-here)，并已正式宣布可用于生产环境！这是多年努力的成果，也是 React Native 生态系统的一个重要里程碑！此时，您可能已经听说过 JSI、Fabric、Turbomodules、同步低开销无桥通信、互操作模式等。这不仅关乎性能，还解决了重要的 UX/DX 问题，例如能够同步测量视图以定位工具提示，或让 `useLayoutEffect` 像在 Web 上一样工作。最让我兴奋的可能是最被低估的一点：React Native 终于可以使用 React 18+ 的并发功能，特别是 transitions、Suspense 以及即将推出的功能。
