## 📱 React-Native {#react-native}

[![React Native 0.79](/emails/issues/229/react-native-0.79.png)](https://reactnative.dev/blog/2025/04/08/react-native-0.79)

**[React Native 0.79 – 更快的工具链及其他改进](https://reactnative.dev/blog/2025/04/08/react-native-0.79)**

近期 React Native 核心团队采用了更频繁的发布周期，虽然本次是小版本更新，但在性能优化方面仍令人振奋。主要亮点包括：

- Metro：冷启动速度提升 3 倍，支持包导出
- JavaScriptCore (JSC) 将迁移至社区包
- iOS：支持 Swift 兼容的原生模块注册
- Android：跳过 JS 包压缩加速应用启动
- 移除远程 JS 调试功能
