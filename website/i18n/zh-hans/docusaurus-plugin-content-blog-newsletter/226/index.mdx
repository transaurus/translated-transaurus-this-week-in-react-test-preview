[**Parcel v2.14.0**](https://parceljs.org/blog/v2-14-0/)

Parcel是个有趣的存在：它是功能完备的强大打包工具。虽未被主流React框架采用/支持，但v2.14.0仍推出了重大更新：

- Server Components with client, server, and static rendering
- Integrated MDX support (using the rust lib `mdx-rs`)
- A `create-parcel` CLI that can be a replacement for `create-react-app`
- A new React error overlay

## 📱 React-Native {#react-native}

[![React Native Worklets RFC](/emails/issues/226/worklets.jpg)](https://github.com/software-mansion/react-native-reanimated/discussions/7264)

[**React Native Worklets RFC**](https://github.com/software-mansion/react-native-reanimated/discussions/7264)

React Native Worklets 是从 Reanimated 中提取的新库，它实现了 React Native 中真正的多线程功能。该库允许你在多个运行时上运行 JavaScript 代码，并通过新的内存模型引入 `Serializables`、`Shareables` 和 `Synchronizables` 来处理跨运行时的数据。

通过将多线程与动画解耦，Reanimated 变得更加轻量，这对追求更小包体积的 Web 应用尤其有益。同时，开发者现在可以在不依赖动画库的情况下，将多线程用于非 UI 任务，如异步存储、数据库操作、实时音频处理或 AI 驱动的计算。
