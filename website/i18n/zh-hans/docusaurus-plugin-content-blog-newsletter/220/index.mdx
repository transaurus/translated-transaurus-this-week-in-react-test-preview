## ⚛️ React {#react}

[![Next.js PR - 支持Node.js中间件](/emails/issues/220/nextjs-middleware.jpg)](https://github.com/vercel/next.js/pull/75624)

**[Next.js PR \- 支持Node.js中间件](https://github.com/vercel/next.js/pull/75624)**

A much-awaited feature for the Next.js community. The latest v15.2 canary has a new `experimental.nodeMiddleware` flag letting you opt-in for `runtime: ‘nodejs’`, enabling you to use Node.js APIs in middleware, and remove the constraints of being limited to APIs compatible with the edge runtime.

Vercel中间件也进行了[彻底重构](https://x.com/galstar/status/1886832573482934706)，以支持多运行时环境，并在[Fluid计算](https://vercel.com/blog/introducing-fluid-compute)上运行——这是Vercel运行无服务器函数的一种新型高成本效益方式。
