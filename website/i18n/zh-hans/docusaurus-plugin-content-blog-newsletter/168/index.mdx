## ⚛️ React {#react}

[![TanStack Router 1.0](/emails/issues/168/tanstack-router.jpg)](https://tanstack.com/router/v1)

**[TanStack Router 1.0](https://tanstack.com/router/v1)**

备受期待的 TanStack Router 以稳定的 1.0 版本发布，并附带一个精美的 [🐦 预告片](https://twitter.com/tannerlinsley/status/1738709130213560497)，正好赶上圣诞节。这个新路由器在类型安全、URL 状态管理（搜索参数）和缓存等方面突破了可能的边界。

[SSR 指南](https://tanstack.com/router/v1/docs/guide/ssr) 引用了一个 `@tanstack/react-start` 包，可能是对即将推出的 [🐦 TanStack Start 框架](https://twitter.com/sebastienlorber/status/1742589454563369123) 的预告。

您可能错过的其他近期 TanStack 包：

- 📦 [TanStack Config - 用于发布和维护高质量 JavaScript 包的工具](https://github.com/TanStack/config)
- 📦 [TanStack Store - 与框架无关、类型安全的存储，带有响应式框架适配器](https://tanstack.com/store/latest)
