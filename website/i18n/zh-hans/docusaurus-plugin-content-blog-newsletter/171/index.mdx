## 📱 React-Native {#react-native}

本周的 React Native 新闻由 Benedikt 整理。有反馈、建议或新闻分享？请在 [Twitter](https://twitter.com/bndkt) 上告诉我。

[![Expo SDK 50](/emails/issues/171/expo50.jpg)](https://expo.dev/changelog/2024/01-18-sdk-50k)

**[Expo SDK 50](https://expo.dev/changelog/2024/01-18-sdk-50)**

Expo SDK 50 引入了 Expo 开发者工具插件，新增了 SQLite 和 Camera API，并添加了 `@expo/fingerprint` —— 一种用于确定更新后的 JS 包是否与给定原生构建兼容的新方法。此版本还标志着 Expo 中 [Webpack 支持的终结](https://blog.expo.dev/webpack-support-in-expo-cli-is-now-deprecated-e9831d7eb631)，现在已弃用，转而支持 Metro。快速了解 Expo 50 的最佳且最便捷的方式可能是观看 Simon 在 [YouTube](https://www.youtube.com/watch?v=shaGaiwu4Fg) 上约 15 分钟的快速讲解。

---

[![Expo Router v3](/emails/issues/171/router3.jpg)](https://expo.dev/changelog/2024/01-23-router-3)

**[Expo Router v3](https://expo.dev/changelog/2024/01-23-router-3)**

随着 Expo SDK 50 的发布，Expo Router v3 也已发布。它带来了许多性能和稳定性改进，并提供了更好的文档。最值得注意的是，现在支持通过 API Routes 构建通用服务器端点（实验性功能）。Expo Router 现在成为了首个通用的全栈 React 框架！
