## 📱 React-Native {#react-native}

[![Unistyles 2.0](/emails/issues/167/unistyles20.jpg)](https://www.youtube.com/@notjustdev/live?sub_confirmation=1)

[**欢迎使用 Unistyles 2.0！**](https://www.reactnativecrossroads.com/posts/unistyles-2-preview)

Unistyles 是一个跨平台样式库。它作为 StyleSheet 的超集构建（如果您使用 React Native，您可能熟悉 StyleSheet.create……），类似于 TypeScript 是 JavaScript 的超集。三个月前首次发布，它已在 GitHub 上获得超过 500 颗星，目前正迎来第二个主要版本，包含增强的 TypeScript 类型、不再依赖 React Context，以及全新的 C++ 绑定以提升性能。一个非常有趣的新特性是插件支持的加入，这将可能实现一些有趣的新用例。
