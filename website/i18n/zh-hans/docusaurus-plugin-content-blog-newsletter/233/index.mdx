## 📱 React-Native {#react-native}

[![Expo SDK 53](/emails/issues/233/expo53.jpg)](https://expo.dev/changelog/sdk-53)

**[Expo SDK 53](https://expo.dev/changelog/sdk-53)**

这是次令人印象深刻的Expo版本更新，主要亮点包括：

- 升级至 React Native 0.79、React 19 和 RNW 0.20
- 默认开启新架构（New Arch）
- Android 端默认启用边到边（Edge-to-edge）模式
- 全新 [Expo BackgroundTask](https://expo.dev/blog/goodbye-background-fetch-hello-expo-background-task)，通过 Android `WorkManager` 和 iOS `BGTaskScheduler` 系统 API 实现现代、可靠、节能且面向未来的后台任务运行方案
- 全新 [Expo Maps](https://expo.dev/blog/introducing-expo-maps-a-modern-maps-api-for-expo-developers)（alpha 版）基于 Expo Modules API 构建，采用 SwiftUI 和 Jetpack Compose，暴露平台默认的 Apple Maps 和 Google Maps 组件
- 默认启用 Metro 的 `package.json#exports` 解析功能
- 通过 `npx testflight` 将开发版本部署至 TestFlight
- 使用预构建 Expo 模块将 Android 构建时间缩短最高 25%
- 以及更多功能！

另请参阅 [Expo Router v5](https://expo.dev/blog/expo-router-v5) 相关发布及其亮点：

- 需 React 19 / Expo 53
- 改进开发者体验（DX）、错误屏幕与日志记录，利用 React 所有者堆栈
- 通过 EAS Hosting 在生产环境部署版本化 API 路由
- React 服务端函数（Server Functions）测试版
- 受保护路由

[![游戏开始 - 将 WebGPU 引入 React Native](/emails/issues/233/skia.jpg)](https://www.youtube.com/watch?v=t9t-VXwIc4I)

**[🎥 游戏开始 - 将 WebGPU 引入 React Native](https://www.youtube.com/watch?v=t9t-VXwIc4I)**

在这段长达 37 分钟、充满激动人心公告的视频中，William Candillon 与嘉宾们带我们一窥绘图与动画的未来。在 Shopify 的支持下，他正将现代 WebGPU 支持（Vulkan 和 Metal）引入 React Native。

通过对齐 Web API，可直接复制/粘贴现有 Web 示例（包括 Three.js）并在全平台运行。同时支持 GPU 通用计算（如 AI、TensorFlow.js），借助便捷的底层 TypeGPU 库可更轻松实现。

所有工作旨在创建基于 WebGPU 的全新实验性 Skia Graphite 后端，支持高性能组合与交织 2D/3D 体验（类似 Web），覆盖全平台并完全集成 Reanimated。过程中 Skia 虽未新增功能，但显著提升性能、减少崩溃，并新增支持三大平台：tvOS、macOS 和 Node。继续调试那只章鱼吧 William！
