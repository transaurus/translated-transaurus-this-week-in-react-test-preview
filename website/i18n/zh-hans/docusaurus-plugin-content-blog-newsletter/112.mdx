## React

[**语句与表达式**](https://www.joshwcomeau.com/javascript/statements-vs-expressions/?utm_campaign=This%20Week%20In%20React&utm_medium=email&utm_source=Revue%20newsletter)

Josh Comeau 正在准备他的新 React 课程。他建议重温 JavaScript 基础知识，从而更好地理解 JSX 的工作原理（JSX 仅支持表达式）。关于这个话题，我期待 [do-expressions](https://github.com/tc39/proposal-do-expressions?utm_campaign=This%20Week%20In%20React&utm_medium=email&utm_source=Revue%20newsletter) 提案能取得进展 😇。

[**如何在 Storybook 中构建连接组件**](https://storybook.js.org/blog/how-to-build-connected-components-in-storybook/?utm_campaign=This%20Week%20In%20React&utm_medium=email&utm_source=Revue%20newsletter)

Storybook 官方文章，详细解释了如何使用装饰器：添加布局、全局提供者、模拟 Redux 存储或请求... 这有助于将原本因上下文依赖而无法渲染的"连接"组件添加到 Storybook。

[**如何估算 NPM 包市场份额（Redux 与其他库的使用情况对比）**](https://blog.isquaredsoftware.com/2022/07/npm-package-market-share-estimates/?utm_campaign=This%20Week%20In%20React&utm_medium=email&utm_source=Revue%20newsletter)

Mark Erikson（Redux）解释了他如何分析npm包的使用趋势，基于各种可靠性不一的工具。下载统计数据通常与CI构建相关。他以React状态管理器为例进行说明，目前估计Redux在React中的使用率为33%（此前为45-50%）。

[**React中的无操作和无状态Reducer**](https://julesblom.com/writing/actionless-and-stateless-reducers?utm_campaign=This%20Week%20In%20React&utm_medium=email&utm_source=Revue%20newsletter)

useReducer()不仅适用于复杂场景：它也可用于非常简单的状态机，比如将布尔值从false切换为true。
