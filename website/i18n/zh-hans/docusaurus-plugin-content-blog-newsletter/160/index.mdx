## 📱 React-Native {#react-native}

[![Expo Router RFC - API Routes](/emails/issues/160/expo-api-routes.jpg)](https://blog.expo.dev/rfc-api-routes-cce5a3b9f25d)

[**Expo Router RFC - API 路由**](https://blog.expo.dev/rfc-api-routes-cce5a3b9f25d)

我一直期待这个功能！Expo 正通过提供基于 Expo Router 约定的 API 路由部署功能，进军服务器端领域。你可以自行部署这些路由，很快也能在流行托管平台上部署，但很明显，其长期目标是建议将服务器运行时托管在 EAS 上。

这标志着 Expo 新时代的开始。在解决了原生移动端的问题后，现在通过服务端渲染和 React 服务组件填补 Web 支持的空白，使跨平台成为主流现实。
