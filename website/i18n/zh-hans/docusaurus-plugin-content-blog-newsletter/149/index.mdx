## ⚛️ React {#react}

[![Panda CSS](/emails/issues/149/panda.jpg)](https://panda-css.com/)

**[Panda CSS](https://panda-css.com/)**

Chakra UI 创始人 Segun Adebayo 正式发布其期待已久的 CSS-in-JS 库（[预告片](https://twitter.com/thesegunadebayo/status/1669732623994834946)）。该库提供卓越的开发体验，基于现代 CSS 特性（级联层），性能优异（构建时静态分析，无运行时开销，原子化 CSS）... 简言之，它满足了众多需求并引发高度关注。尤其与 React 深度集成（JSX 样式属性、兼容 React 服务器组件等），同时也支持其他前端框架。

这是一个值得密切关注的库，有望成为 Tailwind 的有力竞争者：它能更好地与目标框架集成，且无需使用冗长的 `className`。

## 📱 React-Native {#react-native}

[![RN 0.72](/emails/issues/149/rn.jpg)](https://reactnative.dev/blog/2023/06/21/0.72-metro-package-exports-symlinks)

**[React Native 0.72 - 符号链接支持、错误改进及更多](https://reactnative.dev/blog/2023/06/21/0.72-metro-package-exports-symlinks)**

经过多个Golden候选版本后，React-Native的最新版本终于发布，并将很快用于Expo 49 SDK。本次更新包含大量改进（开发者体验、Hermes引擎、性能等）以及部分破坏性变更。我特别欣赏Metro打包工具中的两项新功能：

- 符号链接（测试版）：期待已久的功能，可简化monorepo和pnpm支持
- [包导出](https://reactnative.dev/blog/2023/06/21/package-exports-support)（测试版）：更好的库封装能力，提升与前端生态兼容性，并能暴露特定React Native入口点
