## ⚛️ React {#react}

[![React 19](/emails/issues/213/react.jpg)](https://react.dev/blog/2024/12/05/react-19)

**[React 19](https://react.dev/blog/2024/12/05/react-19)**

过去一年React 19候选版本一直是本通讯的主角，因此除了两项新特性：导致版本延迟发布的预热功能(Pre-warming)和新的React-DOM静态API外，已无需赘述。以下是已覆盖特性的回顾：

- Actions/服务端Actions/`useActionState()`
- `<form>` + 渐进增强兼容的hooks
- 使用`<Context>`替代`<Context.Provider>`
- `use()`钩子
- 自定义元素支持
- 将ref作为标准props及ref回调清理函数
- 元标签处理
- 资源预加载
- 更完善的错误处理及水合差异报告
