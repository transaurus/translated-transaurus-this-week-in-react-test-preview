## ⚛️ React 相关 {#react}

[![2024 年 React 状态 - 调查结果](/emails/issues/221/state-of-react.png)](https://2024.stateofreact.com)

**[2024 年 React 状态 - 调查结果](https://2024.stateofreact.com)**

2024 年 React 生态调查结果已发布。这是座信息金矿，我们无法在此详述所有内容，但以下是几个亮点：

- 在"新" React 功能中，服务器组件最受欢迎，约 7000 名受访者中 40% 表示"使用过"，这可能要归功于 Next.js...
- `forwardRef` 是最不受欢迎的 React API，即将被淘汰！
- 您听说过 React 的 Taint API 吗？约 80% 的 React 开发者表示不了解，这可以理解因为它仍处于实验阶段。
- 榜首位置变化不大：Next.js、TanStack Query、Axios、Jest 和 Zod 仍然是最常用的工具。
- 几颗冉冉升起的新星：Zustand 和 Vitest 用户增长明显，TanStack Start 虽然几乎无人使用过，但关注度排名第一。

Seb 撰写了[调查结论](https://2024.stateofreact.com/en-US/conclusion/)——务必查看哦！🙂
