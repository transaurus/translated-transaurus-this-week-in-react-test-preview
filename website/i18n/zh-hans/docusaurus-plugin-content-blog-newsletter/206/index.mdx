## ⚛️ React {#react}

[![Next.js 15](/emails/issues/206/nextjs-15.png)](https://nextjs.org/blog/next-15)

[**Next.js 15**](https://nextjs.org/blog/next-15)

恰逢 Next.js 大会，Next.js 15 正式发布。自 RC2 以来无重大变更，仅进行了稳定性修复。本版本特性：

- 缓存行为反转：默认不再缓存多数内容（如 `fetch` 请求）
- App Router 使用 React 19 RC，Pages Router 仍基于 React 18
- 现推荐使用 turbopack 开发（仍支持 webpack）
- 存在破坏性变更（如现有API异步化），可通过 `@next/codemod upgrade` 辅助升级

当然还包括大量开发者体验、安全性和性能改进。
