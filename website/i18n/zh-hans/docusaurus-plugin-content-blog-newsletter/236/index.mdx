## ⚛️ React {#react}

[![Remix 觉醒计划](/emails/issues/236/remix.jpg)](https://remix.run/blog/wake-up-remix)

**[Remix 觉醒计划 - v3 版本路线图](https://remix.run/blog/wake-up-remix)**

本周你可能听闻了 [Remix 将放弃 React](https://x.com/AdamRackis/status/1925646909345058818) 的传闻。Ryan 和 Michael 刚刚发布了官方文章阐述 Remix 的未来规划。

简言之，React Router 将继续获得完整维护且稳定可靠。同时，Remix v3 将成为基于 Preact 分支的新框架，实现全栈自主控制。

Remix v3 的目标是"重新构想 Web 框架的可能性"。文章阐述了新框架将遵循的核心原则（为 LLM 优化、基于 Web API 构建、组件化等）。
