## ⚛️ React {#react}

[![React 19.1](/emails/issues/228/react_19_1.jpg)](https://github.com/facebook/react/releases/tag/v19.1.0)

**[React 19.1](https://github.com/facebook/react/releases/tag/v19.1.0)**

React 19.1 已于本周发布，最值得关注的更新包括：

- 新增 [`captureOwnerStack()`](https://react.dev/reference/react/captureOwnerStack) API，框架将借此在开发环境提供更精准的错误追踪（与现有组件堆栈在处理 `children` 和 DOM 节点的方式上存在差异）
- Parcel 工具链中服务器组件支持的 "React 端" 实现，与 Parcel 2.14 新特性深度协同
- 实验性 `unstable_prerender()` 方法，用于在服务端预渲染 React 服务器组件

此版本还包含大量细微改进和修复，多数涉及服务器组件或 `Suspense` 边界处理。
