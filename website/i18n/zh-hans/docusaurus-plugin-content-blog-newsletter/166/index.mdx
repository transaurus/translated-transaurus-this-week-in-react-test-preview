## ⚛️ React {#react}

[![Waku](/emails/issues/166/waku.jpg)](https://waku.gg/blog/introducing-waku)

[**Waku - 增强React Server Component使用的极简框架**](https://waku.gg/blog/introducing-waku)

我们曾介绍过此框架，现在v0.18重写版已发布，包含简介博文。
Waku是极简的React框架，专注于探索核心RSC能力，摆脱传统SSG/SSR策略。技术上它是基于Vite的打包工具，输出纯JS并支持边缘运行时部署。现由Vercel赞助。
