随着 Facebook 和 Twitter 最近的生产部署，我认为一种新的趋势正在逐渐兴起：原子化 CSS-in-JS。

在本文中，我们将探讨什么是原子化 CSS，它如何与功能优先/实用优先的 CSS（如 TailwindCSS）相关联，以及大型企业如何在其现代 React 代码库中采用这种技术。

由于我并非该领域的专家，请勿期待对利弊的深入分析。我只希望您能借此了解其基本概念。

**注意**：原子化 CSS 与 [Atomic Design](https://atomicdesign.bradfrost.com/) 并无实质关联。
