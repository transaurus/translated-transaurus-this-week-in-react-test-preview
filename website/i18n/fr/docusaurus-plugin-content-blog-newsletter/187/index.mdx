## ⚛️ React {#react}

[![Next.js 15 RC](/emails/issues/187/nextjs.jpg)](https://nextjs.org/blog/next-15-rc)

**[Next.js 15 RC](https://nextjs.org/blog/next-15-rc)**

Une nouvelle version majeure de Next.js est sortie en RC lors de Vercel Ship la semaine dernière. Les points forts de cette version sont :

- Support de React 19 RC
- Support expérimental du React Compiler via le plugin Babel (peut augmenter le temps de build)
- Améliorations des erreurs d'hydratation affichant les différences
- Moins de cache agressif, désormais optionnel : `fetch` et les gestionnaires de route ne sont plus mis en cache par défaut
- Adoption progressive du Prerendering Partiel avec l'option `experimental_ppr` dans la configuration de route
- Nouveau design pour `create-next-app`, proposant Turbo en dev et création de projets
- next/after (expérimental), nouvelle API utile pour reporter les tâches analytiques et garder des réponses rapides
- Optimisation du bundle des packages externes (`serverExternalPackages`) désormais stable

D'autres éléments intéressants ont été annoncés à Vercel Ship. Ressources supplémentaires à consulter :
