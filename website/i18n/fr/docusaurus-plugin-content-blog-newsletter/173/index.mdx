## 📱 React-Native {#react-native}

Cette section est désormais co-écrite avec [Benedikt](https://twitter.com/bndkt), et nous avons également un nouveau [fil React Native sur Twitter](https://twitter.com/bndkt/status/1755203992039424377) ! Dites-nous si vous l'appréciez, et soutenez-nous avec un like ou un retweet 😍.

[![Annonce de React Native pour Apple Vision Pro](/emails/issues/173/visionos.jpg)](https://www.callstack.com/blog/announcing-react-native-for-apple-vision-pro)

**[Annonce de React Native pour Apple Vision Pro](https://www.callstack.com/blog/announcing-react-native-for-apple-vision-pro)**

La semaine dernière, Apple a lancé son Apple Vision Pro, et au moins une partie de la communauté React Native est enthousiaste à l'idée d'une nouvelle plateforme pour laquelle ils pourront créer des expériences avec leur framework préféré. Il existe deux approches différentes (🐦 [comparaison côte à côte](https://twitter.com/o_kwasniewski/status/1754520654219493408)) pour intégrer React Native à visionPro :

- Comme vous pouvez exécuter une application iOS sur macOS, vous pouvez cocher une case dans App Store Connect et activer votre application pour qu'elle s'exécute sur visionOS « tel quel », en utilisant le mode de compatibilité. Découvrez comment 🐦 [Evan Bacon provisionne, construit et installe une application Expo sur visionPro en moins de 10 minutes](https://twitter.com/Baconbrix/status/1753927218697208102). Les applications en mode de compatibilité ne tirent cependant pas parti des capacités de « calcul spatial » de visionOS, elles ne sont qu'un écran iPad affiché dans la pièce.

- Pour créer des applications véritablement optimisées pour ce nouveau monde, des capacités natives supplémentaires doivent être adaptées. Et c'est exactement ce que fait cette nouvelle plateforme visionOS, dirigée par Oskar Kwaśniewski et proposée par Callstack.

Ce n'est que le début, mais c'est très excitant de voir RN adopter une nouvelle plateforme aussi rapidement ! La toute première application React Native visionOS [Spacial Noise](https://apps.apple.com/us/app/spatial-noise/id6477335349) est déjà disponible sur l'App Store.
