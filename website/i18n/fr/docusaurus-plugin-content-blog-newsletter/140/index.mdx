## ⚛️ React {#react}

[![React Labs](/emails/issues/140/reactlabs.jpg)](https://react.dev/blog/2023/03/22/react-labs-what-we-have-been-working-on-march-2023)

**[React Labs : Ce sur quoi nous avons travaillé – Mars 2023](https://react.dev/blog/2023/03/22/react-labs-what-we-have-been-working-on-march-2023)**

Une mise à jour officielle de la R&D autour de React qui aborde différents sujets :

- Composants Serveur React. Une belle implémentation est déjà disponible dans Next.js via App Router. Les RSC avec async/await devraient également pouvoir fonctionner dans une application client. J'ai l'impression qu'ils veulent unifier les modèles côté client/serveur et je pense que [AsyncContext](https://twitter.com/sebastienlorber/status/1639183216799100928) sera génial pour ça !
- Chargement des ressources : intégration avec Suspense pour charger CSS, polices et images ([plus de détails](https://twitter.com/acdlite/status/1637102880850059264)).
- Métadonnées du document : support natif d'une API pour les métadonnées `<head>` avec prise en charge du streaming.
- Compilateur d'optimisation React : présenté non plus comme un compilateur de mémoïsation automatique, mais comme un compilateur de réactivité. [React Forget a atteint une étape importante](https://twitter.com/en_JS/status/1641494974893281282), et ils se concentrent sur l'intégration.
- Rendu hors écran : une innovation que j'attends vraiment, qui différencie React des autres frameworks. Meta teste le prérendu sur des apps React Native avec des résultats positifs.
- Traçage des transitions : permet de détecter les transitions lentes.

## 📱 React-Native {#react-native}

- 📜 [Storybook pour React Native (6.5)](https://storybook.js.org/blog/storybook-for-react-native-6-5/) : excellente nouvelle pour les utilisateurs de Storybook sur React-Native, l'intégration rattrape celle de Storybook web, adoptant CSF, Controls, Args, et devrait bientôt être compatible avec Storybook 7.0. Grâce aux nouvelles fonctionnalités de Metro (`require.context`), la nécessité de générer un index d'histoires devrait bientôt disparaître.
- 📜 [Comment la migration de Redux vanilla à Redux Toolkit a amélioré la gestion d'état dans Shopify POS](https://shopify.engineering/react-redux-toolkit-migration) : retour d'expérience sur une migration progressive vers RTK, supprimant 3500 lignes.
- 👀 [RFC : API de traversée du DOM et de mise en page dans React Native](https://github.com/react-native-community/discussions-and-proposals/pull/607) : une nouvelle API avancée pourrait permettre la mise en œuvre de fonctionnalités comme `MutationObserver`, `IntersectionObserver`...
- 👀 [Aplatissement de vues sur iOS](https://twitter.com/SamuelSusla/status/1641086480675225601) : React-Native pouvait déjà optimiser sur Android en évitant de créer des vues natives inutiles. Samuel explique son fonctionnement et annonce que cela fonctionne désormais aussi sur iOS.
