Bonjour à tous !

Ce fut une semaine plutôt excitante en termes d'actualités !

Nous avons eu plusieurs excellentes versions et articles sur React. C'est formidable de voir React-Forget déployé progressivement sur Instagram. Remotion a également démontré tout son potentiel avec le lancement de GitHub Unwrapped.

React-Native est en feu, avec la version stable v0.73, Expo SDK 50 et Expo Router v3 en bêta, ainsi que d'autres excellentes versions. Ce qui m'enthousiasme le plus, c'est qu'Expo dévoile progressivement son plan pour prendre en charge les Universal React Server Components.

Ce numéro est très spécial pour moi : c'est la première fois que je ne rédige pas seul le contenu. [Benedikt](https://twitter.com/bndkt) a presque écrit toute la section React-Native. Je n'ai pas utilisé React-Native en production depuis 2020 😅, c'est donc agréable d'avoir quelqu'un qui utilise [toutes les librairies de pointe](https://github.com/bndkt/sharemystack) pour m'aider à fournir des insights pertinents.

N'oubliez pas de participer aux enquêtes [State of JavaScript](https://stateofjs.com/en-US) et [State of React-Native](https://stateofreactnative.com/). Et si vous pouviez me mentionner à la fin, ce serait cool ! [Theo T3 vous montre comment faire](https://youtube.com/clip/UgkxDdNASo6xNS710ODcjMx0WW4HtTxIYbrA?si=G0rAnDKOQwHqh_ED) 😄

## ⚛️ React {#react}

[![Waku](/emails/issues/166/waku.jpg)](https://waku.gg/blog/introducing-waku)

[**Waku - Framework React minimaliste qui optimise l'utilisation des React Server Components**](https://waku.gg/blog/introducing-waku)

Nous l'avons déjà présenté, mais la réécriture v0.18 est disponible et inclut un petit article d'introduction.
Waku est un framework React minimaliste qui permet d'explorer les capacités fondamentales des RSC, sans les stratégies traditionnelles de SSG/SSR. Techniquement, c'est un bundler basé sur Vite qui produit du JS pur et prend en charge le déploiement sur des runtimes edge. Maintenant sponsorisé par Vercel.
