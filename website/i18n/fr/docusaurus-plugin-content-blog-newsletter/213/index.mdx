## ⚛️ React {#react}

[![React 19](/emails/issues/213/react.jpg)](https://react.dev/blog/2024/12/05/react-19)

**[React 19](https://react.dev/blog/2024/12/05/react-19)**

Les versions candidates de React 19 ont été l'un des sujets principaux de cette newsletter au cours de l'année écoulée, donc il n'y a pas grand-chose de plus à dire, si ce n'est qu'elle est officiellement stable, à part 2 nouvelles fonctionnalités : la fonction de préchauffage qui a retardé la sortie, et les nouvelles API statiques de React-DOM. Voici un bon rappel des autres fonctionnalités que nous avons déjà couvertes :

- Actions / Actions serveur / `useActionState()`
- `<form>` + hooks compatibles avec l'amélioration progressive
- `<Context>` au lieu de `<Context.Provider>`
- Le hook `use()`
- Éléments personnalisés
- Ref comme props standard et fonctions de nettoyage des ref callbacks
- Gestion des balises meta
- Préchargement des ressources
- Meilleures erreurs et reporting des différences d'hydratation
