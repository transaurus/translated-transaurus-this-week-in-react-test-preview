## ⚛️ React {#react}

[![Next.js 15](/emails/issues/206/nextjs-15.png)](https://nextjs.org/blog/next-15)

[**Next.js 15**](https://nextjs.org/blog/next-15)

Sorti juste avant Next.js Conf, Next.js 15 est officiellement publié. Aucun changement majeur depuis RC2, seulement des corrections de stabilité. Dans cette v15 :

- Le comportement de cache est inversé : la plupart des éléments (ex: requêtes `fetch`) ne sont plus cachés par défaut
- Le routeur App utilise React 19 RC, le routeur Pages reste sur React 18
- Turbopack pour le dev est maintenant recommandé (webpack toujours supporté)
- Changements cassants comme des APIs devenant asynchrones, facilités par `@next/codemod upgrade`

Bien sûr, une multitude d'autres améliorations DX, sécurité et performances.
