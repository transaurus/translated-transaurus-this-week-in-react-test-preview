[**TanStack Query v5**](https://tanstack.com/blog/announcing-tanstack-query-v5)

Après de nombreuses pré-versions, TanStack Query v5.0 stable est sorti, incluant le package React Query. En particulier, il comprend :

[**Rspress v1.0 - Un générateur de site statique rapide basé sur Rspack**](https://github.com/orgs/web-infra-dev/discussions/5)

[![Analyse statique des composants React pour le plaisir et le profit](/emails/issues/163/react-forget.jpg)](https://www.youtube.com/watch?v=kjOacmVsLSE)

🎥 [**Analyse statique des composants React pour le plaisir et le profit**](https://www.youtube.com/watch?v=kjOacmVsLSE)

Sathya travaille sur React-Forget, le compilateur tant attendu. Cette conférence révèle des détails intéressants :

- La compilation est rendue possible grâce au nouveau modèle de React, qui n'utilise plus de classes ni de `this`
- Il raisonne localement, peut optimiser en parallèle et être adopté de manière incrémentielle
- Il fonctionne avec du JavaScript non typé, et comprend les conventions React comme les hooks et le préfixe "use"
- Son système d'analyse d'alias est encore plus complet que celui du moteur V8

Sathya indique que nous devrions en entendre davantage sur React-Forget dans les semaines à venir, notamment lors de la conférence ["Understanding Idiomatic React" à React Advanced](https://reactadvanced.com/schedule-offline) dans seulement 2 jours. Bref, nous avons hâte d'en savoir plus !
