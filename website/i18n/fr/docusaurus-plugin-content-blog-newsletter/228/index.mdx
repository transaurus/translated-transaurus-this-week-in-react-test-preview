## ⚛️ React {#react}

[![React 19.1](/emails/issues/228/react_19_1.jpg)](https://github.com/facebook/react/releases/tag/v19.1.0)

**[React 19.1](https://github.com/facebook/react/releases/tag/v19.1.0)**

React 19.1 a été publié cette semaine, les changements les plus notables étant :

- Une nouvelle API [`captureOwnerStack()`](https://react.dev/reference/react/captureOwnerStack) qui sera utilisée par les frameworks pour fournir de meilleures informations d'erreur en développement. Elle diffère des piles de composants existantes dans sa gestion des `children` et des nœuds DOM.
- Le côté "React" de la prise en charge des Server Components dans Parcel, fonctionnant de concert avec les nouvelles fonctionnalités de Parcel 2.14.
- Un `unstable_prerender()` expérimental pour le prérendu des Server Components React sur le serveur.

Cette version inclut également une multitude de changements mineurs et correctifs, principalement liés aux Server Components ou aux limites `Suspense`.
