## ⚛️ React {#react}

[![Next.js 15.3](/emails/issues/230/nextjs.jpg)](https://nextjs.org/blog/next-15-3)

**[Next.js 15.3](https://nextjs.org/blog/next-15-3)**

Une nouvelle version mineure de Next.js est disponible, avec comme points forts :

- Turbopack pour les builds (alpha) : Des builds de production plus rapides passant 8000+ tests (99%)
- Support communautaire pour Rspack (expérimental) : Bundler alternatif compatible Webpack
- Hook Client Instrumentation : Configuration avancée du monitoring et des analytics
- Hooks de navigation : Contrôle du routage avec `onNavigate` et `useLinkStatus`
- Améliorations du plugin TypeScript : Meilleure prise en charge des gros codebases

C'est formidable de voir Next.js s'améliorer avec un bundling plus rapide via Turbopack pour les builds, et s'ouvrir à Rspack comme alternative à Webpack. Les deux ont des compromis et cas d'usage différents.

- 👀 [React Types PR \- `unstable_Activity`](https://github.com/DefinitelyTyped/DefinitelyTyped/pull/72454): There’s a lot of `<Activity>` activity lately, including TypeScript typedefs added. This upcoming component, previously named `<Offscreen>`, has been [mentioned in React Labs updates since 2022](https://react.dev/blog/2022/06/15/react-labs-what-we-have-been-working-on-june-2022#offscreen), and may soon be ready for prime time? This low-level primitive is particularly exciting for frameworks to fully leverage Concurrent React.
- 👀 [Next.js PR \- Experimental bfcache \- Restore state with `<Activity>`](https://github.com/vercel/next.js/pull/77992): Unsurprisingly, Next.js starts experimenting with the `<Activity>` component above, by implementing a back/forward cache feature that allows pressing the browser back button to instantly transition to the previous page while restoring its former React state. In the future, Next.js (and other React/RN frameworks too) will likely implement speculative pre-rendering: React could start rendering a target page as soon as you hover a link, even before you click it.
- 👀 [React Core PR \- Add Suspensey Images behind a flag](https://github.com/facebook/react/pull/32819): An upcoming React feature could help ensure all images have loaded before transitioning to a new page. The rollout plan is still unclear: it might come in a minor or major.

He also shares the some historical bits and technologies that led to the creation of the Server Components.

- 👀 [React Native Fantom](https://github.com/facebook/react-native/blob/main/packages/react-native-fantom/__docs__/README.md) : Un nouvel outil expérimental de tests d'intégration et de benchmarking est activement développé. C'est un lanceur de tests Jest qui exécute les parties C++ de RN, permettant de tester des éléments tels que les nœuds d'ombrage, le layout, les événements, l'ordonnancement et les mises à jour d'état en C++. Cet outil est principalement utile pour tester les internes de React Native, mais son champ d'application pourrait s'étendre aux tests de code bibliothèque/produit à l'avenir.
- 📖 [Sujets avancés sur le développement de modules natifs](https://reactnative.dev/docs/the-new-architecture/advanced-topics-modules) et [Sujets avancés sur le développement de composants natifs](https://reactnative.dev/docs/the-new-architecture/advanced-topics-components) : Différentes pages de documentation pour des sujets avancés de la Nouvelle Architecture ont récemment été publiées, comme l'utilisation de Swift ou l'appel de commandes natives.
- 📜 [Profilage des internes React Native avec Tracy pour des performances optimales](https://www.callstack.com/blog/profiling-react-native-internals-with-tracy-for-peak-performance) : Parfois le goulot d'étranglement de performance ne vient pas de votre application, mais de RN lui-même. Cette solution de profilage unifiée fonctionne sur toutes les plateformes.
- 📜 [Comment activer les flags de fonctionnalité du cœur React Native sur Android](https://andrei-calazans.com/posts/react-native-android-feature-flags/) : Astuce rapide expliquant comment activer `enableFabricLogs` sur Android.
- 📜 [Vous n'avez peut-être pas besoin de react-native-svg](https://blog.swmansion.com/you-might-not-need-react-native-svg-b5c65646d01f) : Cette bibliothèque est très flexible, mais selon le SVG, elle peut créer des arborescences de composants React très volumineuses. De nombreuses alternatives existent avec différents compromis.
- 📜 [Créer des dégradés magnifiques dans React Native avec Skia](https://www.fabrizioduroni.it/blog/post/2025/04/02/react-native-skia-text-gradient)
- 📦 [Expo SDK 53 bêta](https://expo.dev/changelog/sdk-53-beta) : Le nouveau SDK met à niveau vers React Native 0.79, React 19, active la Nouvelle Architecture par défaut et inclut une tonne de nouvelles fonctionnalités. Nous le couvrirons en détail une fois stable dans \~2 semaines !
