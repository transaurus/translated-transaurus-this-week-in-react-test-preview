## ⚛️ React {#react}

[![State of React 2024 - Résultats de l'enquête](/emails/issues/221/state-of-react.png)](https://2024.stateofreact.com)

**[State of React 2024 - Résultats de l'enquête](https://2024.stateofreact.com)**

Les résultats du State of React 2024 sont disponibles. C'est une mine d'informations et nous ne pouvons pas tout aborder ici, mais voici quelques points marquants :

- Parmi les nouvelles fonctionnalités de React, les Server Components sont les plus populaires avec 40% des ~7000 répondants les ayant "utilisées", probablement grâce à Next.js…
- `forwardRef` est l'API React la moins appréciée, et elle va disparaître !
- Avez-vous entendu parler de l'API Taint de React ? Il semble que 80% des développeurs React ne la connaissent pas, ce qui est compréhensible car elle est expérimentale.
- Peu de changement en première position : Next.js, TanStack Query, Axios, Jest et Zod restent les plus utilisés.
- Quelques étoiles montantes : Zustand et Vitest ont gagné des utilisateurs, TanStack Start est \#1 en intérêt bien que presque personne ne l'ait encore utilisé.

Seb a rédigé la [conclusion de l'enquête](https://2024.stateofreact.com/en-US/conclusion/) — assurez-vous de la consulter ! 🙂

[![Bolt x Expo](/emails/issues/221/bolt-expo.jpg)](https://expo.dev/blog/bolt-expo-integration-announcement)

**Construire des applications avec l'IA**

Nous ne parlons pas trop des LLM dans cette newsletter (nous supposons que vous en entendez déjà suffisamment 😅) mais avec toutes les annonces cette semaine, nous avons pensé que c'était le bon moment pour vous donner des nouvelles.

Actuellement, vous pouvez prévisualiser vos applications générées par IA sur le web, mais vous devez télécharger le code pour aller plus loin. Ils prévoient d'ajouter une intégration avec EAS pour déployer en production à l'avenir.

Quelques concurrents avaient également des annonces à faire :

- Il existe désormais un modèle Expo officiel dans Replit, et [une vidéo de démonstration à regarder](https://www.youtube.com/watch?v=mTm_dCF53qk). Replit dispose d'une application, vous pouvez donc créer des applications depuis une application.
